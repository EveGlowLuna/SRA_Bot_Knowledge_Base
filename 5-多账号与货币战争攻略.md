# 多账号托管与货币战争攻略

## 多账号托管

从0.8.0版本开始，SRA添加了多账号托管相关的配置功能。2.0.0版本之后被简化的操作面板所替代。

### 新建方案

点击操作面板左侧的+按钮，输入方案名称后确认，即可添加新方案。每个配置方案对应一个账号的托管方案。

#### 示例

假设配置A、B、C三个账号托管：

- 账号A使用默认方案Default，到任务标签页完成配置。必须配置：启动游戏（游戏路径、自动登录、账号密码）、结束后（退出当前账号）。不能配置：退出游戏、退出SRA、关机、休眠
- 账号B点击+新建方案如planB，像配置Default一样配置即可
- 账号C同理

将操作面板按钮切换到"启动全部配置"模式，点击即可开始多账号托管。方案执行顺序从上到下。建议为每个方案勾选"总是重新登陆"并填写账号密码。

## 货币战争自定义攻略

SRA的货币战争任务使用预定义的攻略，参阅`tasks/currency_wars/strategies/template.json`。

### 编写攻略

```json5
{
  "title": "攻略模版",
  "description": "这是一个攻略模版",
  "author": "作者",
  "uploader": "可选，默认与作者相同",
  "min_coins": 40,
  "min_level": 6,
  "mid_level": 9,
  "on_field": {
    "飞霄": 3,
    "砂金": 3,
    "椒丘": 1
  },
  "off_field": {
    "貊泽": 1,
    "镜流": 1,
    "托帕&账账": 1
  }
}
```

### 添加攻略

打开`tasks/currency_wars/strategies/`文件夹，将编写好的攻略JSON文件放入。在任务配置界面点击攻略下拉菜单右侧刷新按钮即可看到。

## 扩展功能

### 自动对话

开启后在进入剧情时会自动播放对话、自动选择对话选项。跳过对话需先开启自动对话。扩展功能在开启后会立即执行直到取消勾选。
