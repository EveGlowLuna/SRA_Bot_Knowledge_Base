# SRA通知渠道配置

SRA支持多种通知渠道，在任务完成、失败或出错时推送消息。所有渠道默认关闭，按需开启。所有通知渠道受"允许通知"总开关控制。

## 通知数据包格式

```json
{
  "title": "任务完成提醒",
  "event": "任务完成提醒",
  "result": "success",
  "timestamp": "2025-01-01 12:34:56",
  "message": "您的SRA任务运行完成。"
}
```

| 字段 | 说明 |
|---|---|
| title | 通知标题 |
| event | 触发事件名称，同title |
| result | success或fail |
| timestamp | 格式YYYY-MM-DD HH:MM:SS |
| message | 通知正文 |

## 通知触发时机

- 所有任务正常完成 → result为success
- 单个任务执行失败 → result为fail
- 单个任务抛出异常崩溃 → result为fail

## 邮件通知

0.8.1版本引入。在设置中勾选邮件通知，配置SMTP服务。

### SMTP服务器地址

| 提供商 | SMTP服务器 |
|---|---|
| QQ邮箱 | smtp.qq.com |
| 163邮箱 | smtp.163.com |
| Gmail | smtp.gmail.com |
| Outlook/Hotmail | smtp-mail.outlook.com |
| Yahoo Mail | smtp.mail.yahoo.com |

### 授权码

授权码是替代邮箱密码的第三方客户端登录专用密码。

- QQ邮箱：登录后进入账号与安全中心，开启SMTP/IMAP服务获取授权码
- 163邮箱：登录后进入设置→POP3/SMTP/IMAP，开启IMAP/SMTP服务
- Gmail：开启两步验证，然后创建应用专用密码
- Outlook/Hotmail：开启双重验证，然后创建应用密码
- Yahoo Mail：在账号安全中生成应用密码

授权码数据使用Windows DPAPI加密，只有同一用户在同一台计算机上才能解密。

## Webhook

勾选启用Webhook，填写HTTP端点地址。SRA以Content-Type: application/json发送POST请求。

## Telegram

勾选启用Telegram通知，配置：
- Bot Token：通过@BotFather创建机器人获取
- Chat ID：通过@userinfobot获取
- 代理：无法直连Telegram API时配置
- 自定义API地址：自建Bot API服务器时填写
- 发送截图：通知时附带游戏截图

## ServerChan

Server酱将消息推送到微信。勾选启用，填写SendKey（支持SCT和sctp格式）。

## OneBot

OneBot是统一聊天机器人接口标准，用于QQ机器人（如NapCat、LLOneBot等）。

配置：
- Endpoint：OneBot HTTP API地址（如http://127.0.0.1:3000）
- 用户ID：接收私聊的QQ号（与群号至少填写一个）
- 群号：接收群消息的群号
- Access Token：API鉴权Token
- 发送截图

### 以NapCat为例

1. 参考NapCat官方文档完成安装并登录QQ账号
2. 在NapCat网页控制台→网络配置→添加HTTP服务器
3. 记录端口号（默认3000），可选填写Access Token
4. 在SRA中填写Endpoint和用户ID

建议本地部署机器人，Endpoint填http://127.0.0.1:端口号即可。

## Bark

Bark是iOS推送通知应用。配置：
- 设备Key：从Bark App首页复制
- Bark API端点：默认https://api.day.app
- 推送级别：active、timeSensitive或passive
- 通知声音：自定义铃声
- 通知分组：默认StarRailAssistant

## 飞书

支持两种方式：
1. 群机器人Webhook（推荐）：在飞书群设置添加自定义机器人获取Webhook地址
2. 应用API：需配置AppId、AppSecret和接收者ID

两种方式同时填写时优先使用应用API。

## 企业微信

勾选启用，填写群机器人Webhook地址。支持发送截图。

## 钉钉

勾选启用，填写Webhook地址和加签密钥。注意安全设置只选择"加签"，不要同时勾选其他选项。

## Discord

勾选启用，填写频道Webhook地址。支持发送截图。

## xxtui

xxtui是聚合推送平台，支持转发到微信公众号、企业微信等。配置API密钥、通知来源和通知渠道。

## 截图功能说明

支持发送截图的渠道（Telegram、OneBot、企业微信、Discord）有独立的"发送截图"开关。任务通知会截取游戏画面随通知发送。测试通知时发送SRA图标避免截取无关内容。
