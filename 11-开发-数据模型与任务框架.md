# 数据模型与任务框架

## operators.model 模块

定义屏幕自动化操作中的数据模型，包括Region和Box数据类。

### Region

表示屏幕上的一个矩形区域。

```python
@dataclasses.dataclass
class Region:
    left: int
    top: int
    width: int
    height: int
    parent: 'Region | None' = None
```

**属性：** left(左侧坐标)，top(顶部坐标)，width(宽度)，height(高度)，parent(父区域引用)

**方法：**
- tuple属性：将Region转换为(left, top, width, height)元组
- sub_region(from_x, from_y, to_x, to_y)：获取当前Region内的子区域，参数为比例(0-1)

### Box

表示窗口内的一个矩形区域。

```python
@dataclasses.dataclass
class Box:
    left: int
    top: int
    width: int
    height: int
    source: str | None = None
```

**属性：** left(左侧坐标)，top(顶部坐标)，width(宽度)，height(高度)，source(来源，模板路径或文本)

**方法：**
- center属性：获取Box中心点坐标(center_x, center_y)
- distance(other)：计算两个Box中心点之间的欧几里得距离

### Waitable类型别名

```python
type Waitable = Callable[[], Box | None]
```

## task 模块

定义任务框架，提供任务基类、注册机制和生命周期管理。

### Executable

可执行对象的基类，提供停止功能。

```python
class Executable:
    def __init__(self, operator: IOperator):
        self.operator = operator
        self.settings = operator.settings
        self.stop_event = self.operator.stop_event
```

**属性：** operator(操作器实例)，settings(设置对象)，stop_event(停止事件)

**方法：** stop() — 停止执行

### BaseTask

基础任务类，所有任务类都应继承自此。继承自Executable。

```python
class BaseTask(Executable, ABC):
    def __init__(self, operator: IOperator, config: TasksConfig):
        super().__init__(operator)
        self.config = config
        self._post_init()
```

**初始化参数：** operator(操作器实例)，config(任务配置)

**属性：** config(任务配置)

**方法：**
- _post_init()：子类可重写以进行额外初始化
- start() (final)：启动任务，调用on_start()
- run() (abstract)：任务主要执行逻辑，子类必须实现，返回bool
- finish() (final)：完成任务，调用on_finish()
- fail() (final)：任务失败，调用on_failure()
- send_notification(message, result)：发送通知
- on_start()：任务开始时的回调
- on_finish()：任务完成时的回调
- on_failure()：任务失败时的回调

### 任务注册机制

#### registry

全局任务注册表，存储已注册的任务类及其执行顺序。

```python
registry: list[tuple[int, type[BaseTask]]] = list()
```

#### task()装饰器

将任务类注册到全局任务列表，指定执行顺序。

```python
def task(_cls: type[BaseTask] | None = None, *, order: int | None = None)
```

#### get_task_classes()

获取已注册的任务类列表，按执行顺序排序。

```python
def get_task_classes() -> list[type[BaseTask]]
```

### 使用示例

```python
from SRACore.operators.ioperator import IOperator
from SRACore.operators.model import Box
from SRACore.task import BaseTask, task

@task(order=1)
class MyTask(BaseTask):
    def run(self) -> bool:
        # 任务逻辑
        return True

op = MyOperator()
box = op.wait_img("button.png", timeout=10)
if box:
    op.click_box(box)
result = op.ocr_match("Hello World")
box = op.wait_ocr("Loading...")
```
